BASIC USAGE
================================================================================

Python - MATLAB Mode (Default):
-------------------------------
from voice_analysis.core import VoiceAnalyzer
import soundfile as sf

audio, fs = sf.read('voice.wav')
analyzer = VoiceAnalyzer()  # MATLAB mode (152 measures)
measures, F0 = analyzer.analyze(audio, fs)

Python - Thesis Mode:
--------------------
analyzer = VoiceAnalyzer(use_thesis_mode=True)  # Thesis mode (158 measures)
measures, F0 = analyzer.analyze(audio, fs)

# Access thesis-specific measures
print(f"AR Jitter: {measures['jitter_PQ_AR']}")
print(f"NMSP: {measures['jitter_NMSP']}")
print(f"F0 Range: {measures['jitter_F0_range']}")

R - reticulate:
--------------
library(reticulate)
va <- import("voice_analysis.core")

# MATLAB mode
analyzer <- va$VoiceAnalyzer(use_thesis_mode=FALSE)

# Thesis mode
analyzer <- va$VoiceAnalyzer(use_thesis_mode=TRUE)

result <- analyzer$analyze(audio, fs)
measures <- result[[1]]

================================================================================
WHAT'S DIFFERENT BETWEEN MODES?
================================================================================

MATLAB Mode (default):
---------------------
• 152 measures (standard set)
• PPE: Natural logarithm
• Shimmer dB: Original formula
• Backward compatible with MATLAB code

Thesis Mode (use_thesis_mode=True):
-----------------------------------
• 158 measures (standard + 6 additional)
• PPE: Semitone scale (12 × log₂(F₀/127))
• Shimmer dB: Corrected formula
• Additional measures:
  - jitter_PQ_AR: AR-based jitter (Equation 3.39)
  - jitter_NMSP: Normalized perturbation (Equation 3.41)
  - jitter_F0_range: Robust F0 range
  - shimmer_PQ_AR, shimmer_NMSP, shimmer_amp_range

================================================================================
WHEN TO USE EACH MODE?
================================================================================

Use MATLAB Mode When:
--------------------
✓ Migrating from existing MATLAB pipelines
✓ Need backward compatibility

✓ Comparing with published MATLAB results

Use Thesis Mode When:
--------------------
✓ Publishing research citing Tsanas (2012) thesis
✓ Parkinson's disease telemonitoring

✓ Want additional AR-based measures
