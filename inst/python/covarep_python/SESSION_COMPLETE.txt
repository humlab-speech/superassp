───────────────────────────────────────────────────────────────────────
KEY INSIGHTS & LEARNINGS
───────────────────────────────────────────────────────────────────────

1. MATLAB's lpc() is subtle - requires windowing before analysis
2. Pre-frame ramps reduce filter edge artifacts significantly
3. Leaky integration ≠ cumsum (critical difference!)
4. Linear-phase FIR filters preserve glottal flow shape
5. Exact parameter formulas matter (fs/1000 vs 2*round(fs/2000))

COMMON PITFALLS AVOIDED:
  ❌ Using np.cumsum() instead of filter(1, [1 -d], x)
  ❌ Using IIR high-pass filter (introduces phase distortion)
  ❌ Forgetting to window before LPC
  ❌ Wrong sign in Levinson-Durbin coefficient update
  ❌ Omitting pre-frame ramp (causes edge artifacts)
