## Key Insights from This Session

### IAIF Implementation Must Include:

1. **Linear-phase FIR high-pass** (not IIR Butterworth)
   ```python
   B = signal.firls(Nfir+1, [0, 40, 70, fs/2], [0, 0, 1, 1], fs=fs)
   ```

2. **Pre-frame ramp** (reduces edge effects)
   ```python
   ramp = np.linspace(-x[0], x[0], preflt)
   x_ramp = np.concatenate([ramp, x])
   ```

3. **Hanning window before LPC**
   ```python
   win = np.hanning(len(x))
   a = _lpc_matlab_style(x * win, order)
   ```

4. **Leaky integration** (not np.cumsum!)
   ```python
   g = signal.lfilter([1], [1, -d], dg)  # d=0.99
   ```

5. **MATLAB-style LPC**
   - Returns [1, a1, a2, ..., an]
   - Use with: y = filter(a, 1, x)

### Common Pitfalls
- ❌ Don't use cumsum for integration
- ❌ Don't forget windowing before LPC
- ❌ Don't use IIR high-pass filters
- ❌ Don't skip pre-frame ramp
- ❌ Don't approximate formulas
