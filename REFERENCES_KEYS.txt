CITATION USAGE GUIDE
-------------------

Format 1: Single hardcoded reference
  @references
  Alku, P. (1992). Glottal wave analysis...

Format 2: Using citation key
  @references
  \insertRef{Alku1992}{superassp}

Format 3: Multiple keys
  @references
  \insertRef{Alku1992}{superassp}
  \insertRef{Kane2013}{superassp}

Format 4: All entries from .bib file
  @references
  \insertAllCited{}
