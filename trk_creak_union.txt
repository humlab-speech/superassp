_D_e_s_c_r_i_p_t_i_o_n:

     The Union Method \insertCiteWhite2022superassp combines two
     complementary approaches for detecting creaky voice: the AM
     (antimode) method based on F0 distribution analysis, and the CD
     (creak detector) method using a trained artificial neural network
     with 36 acoustic features.

_U_s_a_g_e:

     trk_creak_union(listOfFiles, ...)
     
_A_r_g_u_m_e_n_t_s:

listOfFiles: Vector of file paths to audio files

beginTime: Start time in seconds (not yet implemented)

 endTime: End time in seconds (not yet implemented)

windowShift: Frame shift in milliseconds (default 10ms)

  use.am: Use AM (antimode) method for detection

  use.cd: Use CD (trained ANN) method for detection

cd.threshold: Classification threshold for CD method (0-1, default 0.3)

use.reaper: Use REAPER for F0 estimation
          \insertCiteGhahremani2014superassp (requires pyreaper in
          Python)

return.features: Include intermediate feature tracks in output

return.probabilities: Include classification probabilities (not just
          binary decisions)

explicitExt: File extension for output files (default "crk")

outputDirectory: Directory for output files (default NULL = same as
          input)

  toFile: Whether to write results to disk (default TRUE)

conda.env: Conda environment name for Python (default NULL =
          auto-detect)

_D_e_t_a_i_l_s:

     This function uses the ‘av’ package to load audio files in-memory,
     supporting all formats (WAV, MP3, MP4, etc.). It returns
     comprehensive track data including both individual methods (AM and
     CD) and their union, along with intermediate features used in
     classification.

_V_a_l_u_e:

     An AsspDataObj (or list of AsspDataObjs) containing multiple
     tracks:

     AM_creak Binary decisions from AM method (if use.am=TRUE)

     CD_creak Binary decisions from CD method (if use.cd=TRUE)

     union_creak Union of AM and CD decisions

     CD_prob Classification probability from ANN (if
          return.probabilities=TRUE)

     F0 Fundamental frequency track (if use.am=TRUE)

     H2_H1 Harmonic amplitude difference (if return.features=TRUE)

     peak_prominence Residual peak prominence (if return.features=TRUE)

     IFP Intraframe periodicity \insertCiteIshi2008superassp (if
          return.features=TRUE)

     IPS Interpulse similarity \insertCiteIshi2008superassp (if
          return.features=TRUE)

     PwP_rise Power pulse waveform rising \insertCiteIshi2008superassp
          (if return.features=TRUE)

     PwP_fall Power pulse waveform falling \insertCiteIshi2008superassp
          (if return.features=TRUE)

     Attributes include antimode value and method details.
